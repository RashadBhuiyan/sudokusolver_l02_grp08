Sudoku Rules

1 Use Numbers 1-9

Sudoku is played on a 9 x 9 grid, divided into nine 3 x 3 boxes). Each row, column, and box (9 cells each) needs to be filled with a number between 1-9.

Each game starts with a few cells already filled; the more cells are filled, the easier the game.

2 No Repeated Numbers

No digit can appear twice in the same row, column, or box.

As illustrated, the upper left box (circled in blue) has 2 unfilled cells. The only numbers that can be used without repeats are 5 and 6. By applying the same process of elimination for each row, column, and box we can deduce the rest of the unfilled cells.

3 Don't Guess

Sudoku is a game of logic and reasoning, so you shouldn't have to guess. If you don't know what number to put in a certain space, keep scanning the other areas of the grid until you see an opportunity to place a number.

Sudoku rewards patience, insights, and recognition of patterns, not blind luck or guessing.
