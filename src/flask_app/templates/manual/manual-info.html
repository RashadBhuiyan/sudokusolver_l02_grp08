Manual Input Instructions

Step 1 Manual Input

Edit the cells of the board so that it reflects the desired Sudoku board you wish to be solved.

Step 2 Verification

Please ensure that a valid Sudoku board is being submitted as an invalid board would require edits to make it valid.

Press the 'Solve' button to submit the puzzle for automatic solving.

Step 3 Solution

Voila! You can now view the solution.

If a solution is not available, click 'Back' to double check the puzzle to ensure that the puzzle is solvable.
